# Putty

Putty is an exotic options market for NFTs and ERC20s

**Links:**

* [putty.finance](https://putty.finance)
* [testnet.putty.finance](https://testnet.putty.finance)

---

## Installation

```shell
npm install
```

## Testing

```shell
npx hardhat test
```

## Deploy

```shell
npx hardhat node
npx hardhat deploy --network localhost
```

## Docs

```shell
npx hardhat dodoc
```
